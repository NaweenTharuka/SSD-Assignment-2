Fully Loaded - OATH2.0 APP - Google Drive File Upload


    Installation

        *Package Manager, npm
            $ npm install

        *Developmnet Environment   
            $ npm run dev
                # Development stage ( This will run nodemon internally )

            $ npm run prod
                # Production version ( This command will run node index.js internally to perform well in production environment )
